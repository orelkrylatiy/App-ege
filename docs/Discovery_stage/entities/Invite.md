# Invite

**Определение:** Приглашение для регистрации нового ученика. Создаётся
репетитором, содержит уникальный токен (используется в URL ссылки),
имеет ограниченный срок жизни.

## Атрибуты

- `id: string` — уникальный идентификатор
- `token: string` — уникальный токен для URL (например, `abc123xyz`)
- `tutorId: string` — FK на [Tutor](./Tutor.md) (кто пригласил)
- `studentId?: string` — FK на [Student](./Student.md) (кто использовал, заполняется после регистрации)
- `status: InviteStatus` — текущий статус
- `expiresAt: Date` — срок действия
- `createdAt: Date` — дата создания
- `usedAt?: Date` — дата использования (если было)

## Возможные статусы (`InviteStatus`)

- `ACTIVE` — активна, можно использовать
- `USED` — уже использована для регистрации
- `EXPIRED` — истёк срок действия

## Связи

- → [Tutor](./Tutor.md): **many-to-one** (репетитор может выпустить много приглашений)
- → [Student](./Student.md): **one-to-one** (опционально, после использования)

## Заметки

- Полная ссылка для ученика: `https://app.com/invite/{token}`
- Срок жизни предлагаю 7 дней по умолчанию (уточнить позже).
- Когда статус становится `EXPIRED` — решаем позже: отдельный фоновый
  процесс или вычисляем на лету при чтении.
