# Tutor

**Определение:** Репетитор — пользователь системы, который ведёт учеников,
создаёт для них ДЗ и проверяет ответы.

## Атрибуты

- `id: string` — уникальный идентификатор
- `email: string` — для логина
- `passwordHash: string` — хэш пароля (не сам пароль!)
- `firstName: string` — имя
- `lastName: string` — фамилия
- `middleName?: string` — отчество (опционально)
- `bio?: string` — краткое описание о себе
- `createdAt: Date` — дата регистрации

## Связи

- → [Student](./Student.md): **one-to-many** (один репетитор ведёт много учеников)
- → [Assignment](./Assignment.md): **one-to-many** (один репетитор создаёт много ДЗ)
- → [Invite](./Invite.md): **one-to-many** (один репетитор может выпустить много приглашений)

## Заметки

- Список учеников в атрибутах НЕ хранится — связь через `tutorId` в Student.
