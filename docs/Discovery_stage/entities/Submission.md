# Submission

**Определение:** Ответ ученика на конкретное ДЗ. Содержит текстовый ответ
и/или прикреплённые файлы. Создаётся, когда ученик отправляет ДЗ на проверку.

## Атрибуты

- `id: string` — уникальный идентификатор
- `text?: string` — текстовый ответ (опционально)
- `assignmentId: string` — FK на [Assignment](./Assignment.md)
- `createdAt: Date` — дата отправки

## Связи

- → [Assignment](./Assignment.md): **one-to-one** (один ответ относится к одному ДЗ)
- → [File](./File.md): **one-to-many** (к ответу может быть прикреплено несколько файлов)

## Заметки

- Прямая связь со Student не нужна — её можно получить через Assignment.
- В MVP ответ создаётся один раз и не редактируется. В будущем возможна
  пересдача (тогда нужно будет ослабить связь до one-to-many и добавить
  поле `version` или `attemptNumber`).
- Поле `text` опционально, потому что ответ может состоять только из файлов.
